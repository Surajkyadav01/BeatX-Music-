# 🎵 BeatX - Premium Music Streaming Experience 🎧

**BeatX** is a high-performance, sleek, and responsive web-based music player. Built with a focus on premium UI/UX, it allows users to stream millions of songs in real-time and manage their personal cloud collection.

---

## ✨ Key Features

*   🚀 **Real-time Streaming:** Fetches millions of songs, albums, and artists using the JioSaavn API.
*   🔍 **Smart Auto-Suggestion:** Instant search results as you type (powered by Debouncing logic).
*   ☁️ **Personal Cloud:** Integrated with **Firebase Firestore** to upload and manage your own music library.
*   🛡️ **Admin Control:** Secure "Create" section with password protection for adding/deleting cloud tracks.
*   📱 **Fully Responsive:** Smooth experience across Mobile, Tablet, and Desktop.
*   🎨 **Premium UI:** Modern dark theme with glassmorphism effects and neon glows.

---

## 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3 (Custom Animations), JavaScript (ES6+)
*   **Backend/Database:** Firebase Firestore (NoSQL)
*   **API:** Unofficial JioSaavn API (saavn.sumit.co)
*   **Icons:** FontAwesome 6.0
